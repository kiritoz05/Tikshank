# Deploy TikShankz

## Vercel
1. Sube el repo a GitHub.
2. Importa el proyecto en Vercel.
3. Selecciona `apps/web` como Root Directory.
4. Ejecuta el deploy.

## Railway
1. Crea un nuevo proyecto.
2. Conecta tu repo GitHub.
3. Selecciona `apps/api` como servicio.
4. Añade PostgreSQL y Redis.
5. Configura variables de entorno desde `.env.example`.
6. Genera dominio público.
